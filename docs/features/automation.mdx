MindsDB provides mechanisms to automate tasks. These include [jobs](/mindsdb_sql/sql/create/jobs), [triggers](/mindsdb_sql/sql/create/trigger), and [chatbots](/mindsdb_sql/agents/chatbot).

</p>

<Tip>
Use AI automation to keep your AI systems up-to-date by continuously retraining or fine-tuning them with real-time data. Follow [this use case](/sql/tutorials/real-time-trading-forecasts) to learn how to build it with MindsDB.
</Tip>

<Tip>
Use AI automation to create chatbots. Follow [this use case](/sql/tutorials/twilio-chatbot) to learn how to build custom chatbots with MindsDB.
</Tip>
