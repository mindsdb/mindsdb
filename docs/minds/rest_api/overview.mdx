Minds provide REST APIs, enabling interaction with Minds via API endpoints.

Follow these steps to get started:

<Steps>
  <Step title="Create your Minds account">
    Go to [Minds Cloud](https://mdb.ai/register) to create an account.
  </Step>
  <Step title="Get an API key">
    Go to the [API Keys tab](https://mdb.ai/apiKeys) to get an API key.
  </Step>
  <Step title="Use Minds via REST APIs">
    Follow [this tutorial](/minds/rest_api/create_mind) to create a Mind.
  </Step>
</Steps>
