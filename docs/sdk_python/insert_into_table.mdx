## Description

The `query()` function is executed on a data source connected to MindsDB and saved into a variable. It inserts into a table from this data source.

## Syntax

Here is the syntax:

```sql
my_data_source.query('INSERT INTO integration_name.table_name (SELECT ...)')
```
