MindsDB provides a custom syntax to create an agent that comprises an AI model and data used to customize this model. Agents are connected with a chat interface to create chatbots.

See details following [this link for Agents](/mindsdb_sql/agents/agent) and [this link for Chatbots](/mindsdb_sql/agents/chatbot).
