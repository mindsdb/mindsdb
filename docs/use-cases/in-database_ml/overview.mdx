MindsDB brings AI/ML into your database. This mitigates the need for creating and maintaining data pipelines. Access various [AI/ML models](/integrations/ai-overview) directly from your [data source](/integrations/data-overview).

  <img src="/assets/use_cases/indatabase_ml.jpg" />
</p>

This section covers the following use cases:
    
    - Automated classification models
    - Automated regression models
    - Bring Your Own Model (BYOM) to MindsDB
